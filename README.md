# Agenda de contatos em Python: "Este é um sitema simples via linha de comando, feito para gerenciar contatos, permitindo criar, listar, editar, excluir e favoritar informações."

## 2- Funcionalizades do Sistema:
* Salvar contatos (Nome, telefone);
* Listagens enumeradas de todos os contatos salvos;
* Edição de contatos existentes;
* Eclusão de contatos por índice.

## 3- Pré-requisitos para Excecução do Código:
*Python 3.10 ou superior: O projeto utiliza a estrutura `Match/case`.

## 4- Como Rodar:
Este é um passo a passo técnico, porém bem simples para a excecução do programa em sua máquina.
  1- Baixe o arquivo `agenda_contato.py` .
  2- Abra o terminal ou prompt de comando (CMD).
  3- Navege até a pasto do arquivo.
  4- Execute o comando: `python agenda_contato.py`.
  
## 5- Estrutura do Código:
Aqui são algumas estruturas que foram usadas no decorrer do código, para um melhor esclarecimento.
* Listas: para armazenar o conjunto de contatos;
* Dicionário: para estruturar cada contato individualmente com as chaves 'nome' e 'numero';
* Funções: para modularizar as ações (salvar, editar, etc).

## 6- Técnologias Utilizadas:
* Python.
